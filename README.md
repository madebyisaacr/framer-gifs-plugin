# GIFs Framer Plugin

This is a Framer plugin for inserting GIFs into your projects. Powered by [Klipy](https://klipy.com/), the #1 library of free GIFs and stickers.

[Open on Framer Marketplace](https://framer.link/aDGGfVH)
